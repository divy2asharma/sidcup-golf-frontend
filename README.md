# ⛳ Sidcup Golf Website (Frontend)

A modern and responsive **golf club website UI** built using frontend technologies.
This project replicates a real-world website design with smooth animations, clean layout, and responsive design.

---

## 🚀 Features

* 🎯 Responsive design (mobile, tablet, desktop)
* 🎨 Clean and modern UI/UX
* ⚡ Smooth scrolling and animations
* 📱 Fully optimized layout
* 🧩 Reusable components

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
